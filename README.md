# emotion-recognition
### DenseNet vs machine learning based emotional image   
### Datasets:IAPSa   

### 实现：
    UI界面展示图像分类情况（基于训练图像数据集的标签：Amusement、Anger、Awe、Contentment、Disgust、Excitement、Fear、Sad）   
    
    后续可以更换其他模型替代，界面可以展示模型预测准确率，并将错误识别图像名字标红   
    
### 问题：
    训练的数据量较小，对于相似情感会存在一定的误判情况
    模型参数未调整

### 总结：
    缺少现实意义
